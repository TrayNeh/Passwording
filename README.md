# Passwording
Passwording es una herramienta ligera para generar contraseñas aleatorias y seguras. Permite crear contraseñas adaptadas a tus necesidades (longitud y tipos de caracteres) de forma rápida y fiable, ideal para proteger cuentas, servicios y proyectos. Diseñada para ser simple de usar y fácil de integrar en flujos de trabajo y scripts.

* Qué hace: Genera contraseñas aleatorias con entropía adecuada para uso personal y profesional.

* Personalización: Ajusta la longitud y los conjuntos de caracteres (letras mayúsculas/minúsculas, números, símbolos) para cumplir políticas de seguridad.

* Uso sencillo: Pensado para ejecutarse como herramienta de línea de comandos o integrarse en otros programas/scripts.

* Seguro y reproducible: Produce contraseñas con buena aleatoriedad y sin almacenar las contraseñas generadas.

* Casos de uso: creación de contraseñas para cuentas nuevas, gestores de contraseñas, automatización en despliegues o pruebas.
